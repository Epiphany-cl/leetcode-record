## 2. KMP 算法

> KMP算法通过预处理模式字符串以构建前缀函数，从而在匹配过程中高效地跳过不必要的重复比较，确保了线性的时间复杂度。这使得它在处理大规模字符串匹配问题时特别有效。

KMP（Knuth-Morris-Pratt）算法的名字来源于三位计算机科学家：Donald Knuth、Vaughan Pratt 和 James H.
Morris，他们共同在1970年发表了一篇论文介绍了这个算法。
